# Tryagain123456.github.io

Personal GitHub Pages site for Lee Yan.

## Structure

- `index.html`: landing page entry
- `assets/css/`: stylesheets
- `assets/js/`: client-side scripts
- `assets/images/`: public images used by the site
- `assets/files/`: public downloadable files
- `private/`: local-only source materials, excluded from git

## Local preview

Open `index.html` directly in a browser, or serve the repository root with any static file server.

## Deployment

This repository is intended for GitHub Pages user-site deployment with the repository name `Tryagain123456.github.io`.
